TAGS & THEIR USE:


<table>
  • the tags used to create a table are enclosed within <table></table> tags.

<td>
  • table cell & stands for table data.
  • Table data/text is enclosed within <td></td> tags.

<tr>
  • table rows.
  • <tr></tr> tags are used to add a table row and in order to add text into this particular cell you use <td></td> tags.
  • For every <td></td> tags you include within the <tr></tr> tags, it will create a column within that row.

<th>
  • table header
  • The text within the <th></th> tags is bold and centered. However, it can be changed with CSS.

<thead>
  • the header content in a table.
  • every <tr> element within <thead></thead> tags, will be grouped into the header content of the table.

<tbody>
  • the body content in a table.
  • every <tr> element within <tbody></tbody> tags, will be grouped into the body content of the table.

<tfoot>
  • the footer content in a table
  • every <tr> element within <tfoot></tfoot> tags, will be grouped into the footer content of the table.

<caption>
  • table caption.
  • aka the table's title.

<colgroup>
  • a group of one or more columns in a table for formatting.
  • is used along with <col>.

<col>
  • column properties for each column within a <colgroup> element.
  • is used along with <colgroup></colgroup>.


Using <colgroup> and <col>:


To style specific columns of a table, use the <colgroup>s and <col> tags.

⁃ <colgroup> = container for column specifications.
⁃ <col> specifies each group of column/s + is a self-closing tag.
⁃ span specifies number of columns that get the style.
⁃ style specifies the style (CSS).

For example:

<table>
*first column*
<colgroup>
<col span="1" style="background-color: rgb(152, 174, 204);">
</colgroup>

*last 3 columns*
<colgroup>
<col span="4" style="background-color: rgb(34, 65, 91);">
</colgroup>
. . .
</table>

CSS Properties for <col>

These properties are used under the style attribute. When applied to the selected columns using <col>, these properties will only be shown the the chosen columns.

  1. width property
  2. visibility property
  3. background property
  4. border property


Empty Colgroups
If you want to style columns in the middle of the table, insert an empty <col> element without styling it for the preceeding columns.

For example:

<table>
*first column*
<colgroup>
<col span="1">
</colgroup>

*middle 2 columns*
<colgroup>
<col span="2" style="background-color: rgb(34, 65, 91);">
</colgroup>
. . .
</table>

Hiding Colgroups
If you want to hide columns, set the visibility to collapse.

For example:

<table>
*first column*
<colgroup>
<col span="1">
</colgroup>

*middle 2 columns*
<colgroup>
<col span="2" style="visibility: collapse">
</colgroup>
. . .
</table>

Notice how the "Last Name" and "Age" columns aren't visible?

STYLING HTML TABLES USING CSS:


1. Adding a Border


In order to add a border, use the border property on table, th, and td elements.

table, th, td {
border : [thickness of border] [border style] [border color]; }

For example:
table, th, td {
border : 1px solid black;
width: 500px;
height: 50px;
text-align: center;
}

2. Collapsed Table Border


If you want a single bordered table, set the CSS border-collapse property to collapse.

table, th, td {
border: [thickness of border] [border style] [border color];
border-collapse: collapse ;}

For example:
table, th, td {
border : 1px solid black;
border-collapse: collapsed;
width: 500px;
height: 50px;
text-align: center; }

This will create a table like this:

Ali Khan 23

3. Styling Table Borders


Invisible borders


If you want an invisible border table you could set the background-color to the color of choice and set the border color to white.

table, th, td {
border: [thickness of border] [border style] white;
border-collapse: collapse ;}

th, td {
background-color: [color of your choice] ;}

For example:
table, th, td {
border: 1px solid white;
border-collapse: collapse;
}
th, td {
background-color: #96D4D4;
}

This will create a table like this:


Name Last Name Age
Ali Khan 23
Jack Smith 25
Chloe Wright 21


Round Table Borders


To get rounded table corners, use the border-radius property.

table, th, td {
border: [thickness of border] [border style] [border color];
border-radius: [radius of corners] ;}

For example:
table, th, td {
border: 1px solid black;
border-radius: 10px;
width: 500px;
height: 50px;
text-align: center;
}

This will create a table like this:


Name Last Name Age
Ali Khan 23
Jack Smith 25
Chloe Wright 21

To remove the border around the table, exclude table.

For example:
th, td {
border: 1px solid black;
border-radius: 10px;
width: 500px;
height: 50px;
text-align: center;
}

Name Last Name Age
Ali Khan 23
